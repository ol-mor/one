	Основные команды в терминале:-
Переход в  папку: cd ~/Desktop
Cоздание новой папки: mkdir new-directory
Переход в папку : cd new-directory
Создание файла с текстом: echo "Some text" > file.txt
Список файлов в папке: ls
Список файлов в папке включая скрытые: ls -la(ls -Force для Windows)
Чтение файла  cat file.txt
Удаление файла: rm file.txt


Основные команды git
git init - создание нового репозитория( набирать в папке, где хотим создать репозиторий. После инициализации создается скрытая папка .git)

git add(имя файла) - Перемещаем файлы в " Индекс" для подготовки к сохранению в репозитории

git add . Добовляем все изменения в "индекс"

git commit -m"text" сохроняем все изменения и пишем коментарий
git log - показывает все комиты
git chechout - перейти к определенной версии


git branch <branch name> - создание новой ветки
git chechout -b <branch name> создание новой ветки и переход 
в нее
git branch - отоброжает список всех веток
git branch -m <new branch name> - переименовать текущую ветку
git branch -d <branch name> - удалить ветку(текущую ветку удалить нельзя)
git merge <feature branch name> слияние текущей ветки с указанной веткой
git cat-fale -p <имя файла>-чтение из  hash   файла 

        

git push - переместить изменения с локального репозитория в удаленный
git pull переместить изменения из  удаленного репозитория в локальный
git clone <url> Клонирование  удаленного репозитория в локальный
origin - Имя удаленного репозитория по-умолчанию(рекомендуемое)
git branch -a показывает ветки и в удаленном репозитории тоже
